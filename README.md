# hasty-python
Python library for the Hasty API.

## deployement to PyPi (pip)
* python setup.py sdist
* pip install twine
* twine upload dist/*
* [enter username and password]

## installation (github)
* git pull https://github.com/hasty-ai/hasty-python.git
* cd hasty-python
* pip install -e . 

## installation (pip)
* pip install hasty_python
